A simple Django CRUD application using:
 Django
  Django REST Framework
 JavaScript (Fetch API)
 HTML, CSS

 Features
  Create product
 Read products
Update product
  Delete single product
Delete all products

 Frontend: HTML, CSS, JavaScript
